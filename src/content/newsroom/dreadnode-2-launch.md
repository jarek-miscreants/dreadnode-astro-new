Dreadnode 2.0 is the first platform that supports the full security agent
lifecycle. Teams can now build, evaluate, observe, and safely deploy agents
against real security workloads without stitching together point tools.

This release brings production-ready evaluations, deep observability across
tool calls and model invocations, and sandboxed deployment targets — closing
the loop between prototype and production for offensive security engineering.
